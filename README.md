# Mood_Analyzer

Mood Analyzer is a web-based application designed to help users track, analyze, and reflect on their emotional states. The application allows users to select their mood, add additional feelings, and store this information securely in a database. It also provides engaging games and a history feature to view past mood entries.

## Features
- User-Friendly Interface built with HTML, CSS, and JavaScript
- Mood Tracking System to select and save moods with extra notes
- Database Integration using PyMySQL to store mood history
- Interactive Games Section with 5 fun games to improve user engagement
- Mood History Page to view previously saved moods
- Flask Backend for handling routes, logic, and database operations
